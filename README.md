# armbian-monitor
A remote system monitor over SSH for Armbian. To set up, create a `config.txt` file on the same folder as the program .exe, and follow the format:

```
Armbian host IP
SSH username
SSH password
(Optional) Root password
```

e.g.:

```
192.168.1.45
pipe
hunter2
root2
```
